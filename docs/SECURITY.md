# SECURITY

## 기본 원칙
- 토큰/키/비밀번호/개인정보는 .env에만 저장
- 로그/에러/감사 이벤트는 자동 마스킹

## 마스킹 규칙(예시)
- API 키: /sk-[A-Za-z0-9_-]+/ → sk-***
- Bearer 토큰: /Bearer\s+\S+/ → Bearer ***
- 이메일: /[\w.-]+@[\w.-]+/ → ***@***
- IP: /\b\d{1,3}(?:\.\d{1,3}){3}\b/ → ***.***.***.***
- 경로: /[A-Za-z]:\\[^\s]+/ → ***

## Agent 보안
- accessToken TTL 15~60분
- refresh 토큰 필요
- SANDBOX 제한 + 레이트리밋

## 운영
- 신고/차단/레이트리밋 변경은 AuditEvent에 기록
